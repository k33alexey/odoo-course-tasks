# HR Hospital Management System

Модуль для автоматизации работы медицинского центра на базе **Odoo 19.0**. 
Проект разработан с учетом стандартов декомпозиции кода и нормализации именования объектов.

## 📋 Функциональные возможности

Модуль разделен на два основных логических блока:

### 📑 Документы (Documents)
* **Визиты (Visits):** Регистрация приемов, планирование дат, выбор врача и пациента.
* **История болезни:** Ведение заметок врача и учет диагнозов.

### 📚 Справочники (References)
* **Врачи (Doctors):** Реестр персонала с указанием специализации.
* **Пациенты (Patients):** Карточки пациентов с историей посещений и персональными данными.
* **Диагнозы (Diseases):** Классификатор заболеваний с цветовой маркировкой (tags).

## 🛠 Технический стек

* **Backend:** Python (Odoo Framework)
* **Frontend:** Odoo XML (OWL-ready)
* **Database:** PostgreSQL
* **Архитектурный стандарт:** Использование префиксов `model_name_type` для XML ID.
